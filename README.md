# pb-objectex

#### 介绍
PowerBuilder工程的一般由pbl、pbt文件构成。项目中的窗体对象、函数对象、数据容器等pb对象是存储在pbl文件中。PowerBuilder IDE 可以把pbl文件中的pb对象导出成独立的对象文件，但批量导出的功能不太友好。powerbuilder-objectex提供的工具能把pbl文件批量导出pb对象的问题。除了批量导出，还是批量导入对象到pbl文件。
